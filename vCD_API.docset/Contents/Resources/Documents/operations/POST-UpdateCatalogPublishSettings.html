POST /admin/catalog/{id}/action/publish

Operation:
POST /admin/catalog/{id}/action/publish
Description:
Publish a catalog.

Publishing a catalog sets the value of its IsPublished element to true and makes the catalog visible to organization administrators of all organizations in a cloud.

Since:
1.0
Input parameters
Consume media type(s):
application/vnd.vmware.admin.publishCatalogParams+xml
Input type:
PublishCatalogParamsType
Output parameters
Produce media type(s):
None
Output type:
None
