PUT /admin/extension/settings/branding

Operation:
PUT /admin/extension/settings/branding
Description:
Update system branding settings.

You can use system branding settings to customize the vCloud Director Web console and some of the links that appear on the vCloud Director Home login screen.

Since:
1.5
Input parameters
Consume media type(s):
application/vnd.vmware.admin.brandingSettings+xml
Input type:
BrandingSettingsType
Output parameters
{@link JAXBElement}

Produce media type(s):
application/vnd.vmware.admin.brandingSettings+xml
Output type:
BrandingSettingsType
