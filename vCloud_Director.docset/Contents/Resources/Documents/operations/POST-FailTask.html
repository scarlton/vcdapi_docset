POST /admin/extension/blockingTask/{id}/action/fail

Operation:
POST /admin/extension/blockingTask/{id}/action/fail
Description:
Fail a blocking task.

This request ends the task, setting the status of any transient objects that it created to ERROR. Task status is set to ERROR.

1.5
