Operation:
GET /admin/extension/vimServer/{id}/networks
Description:
Retrieve a list of available vCenter networks and portgroups.

Networks and portgroups that are already in use by a vCloud Director network are not listed.

Since:
1.5
Input parameters
Consume media type(s):
None
Input type:
None

Output type:
