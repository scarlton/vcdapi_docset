POST /admin/vdc/{id}/action/disable

Operation:
POST /admin/vdc/{id}/action/disable
Description:
Disable an organization vDC.

Disabling an organization vDC prevents prevent any further allocation of resources to it, but does not affect existing allocations. When an organization vDC is disabled, an organization user cannot deploy new virtual machines in it.
