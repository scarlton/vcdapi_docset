POST /admin/extension/service/{id}/resourceclasses

Operation:
POST /admin/extension/service/{id}/resourceclasses
Description:
Registers resource class for extension service. The resource class will be added only if a resource class with the same name does not exist.
Since:
Input parameters
New resource class parameter

Consume media type(s):
application/vnd.vmware.admin.resourceClass+xml
Input type:
ResourceClassType
Output parameters
Newly registered or already existing resource class.

Produce media type(s):
application/vnd.vmware.admin.resourceClass+xml
Output type:
ResourceClassType
