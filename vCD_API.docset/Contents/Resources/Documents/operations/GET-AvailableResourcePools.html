GET /admin/extension/vimServer/{id}/resourcePoolList

Operation:
GET /admin/extension/vimServer/{id}/resourcePoolList
Description:
Retrieve a list of unused resource pools from a vCenter server.

Resource pools that are already in use by a provider vDC are not listed.

Since:
1.0

Consume media type(s):
