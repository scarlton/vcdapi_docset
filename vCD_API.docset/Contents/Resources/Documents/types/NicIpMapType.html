NicIpMapType

Type:
NicIpMapType
Namespace:
http://www.vmware.com/vcloud/v1.5
Description:
VM's with NICs that are connected directly to a VDC network and where the NICs have manually assigned IP addresses needs to be mapped to a valid IP address within the IP range of the destination site's VDC network. This type provides that mapping.
