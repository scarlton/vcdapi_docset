POST /catalog/{id}/action/sync

Operation:
POST /catalog/{id}/action/sync
Description:
Force sync the catalog to the remote subscribed catalog.
Since:
5.5
Input parameters
Consume media type(s):
None
Input type:
None
Output parameters
TaskType

The response includes a Task element. You can monitor the task to to track the sync of the catalog.

Produce media type(s):
application/vnd.vmware.vcloud.task+xml
Output type:
TaskType
