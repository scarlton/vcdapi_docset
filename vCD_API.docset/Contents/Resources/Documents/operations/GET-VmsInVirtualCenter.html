GET /admin/extension/vimServer/{id}/vmsList

Operation:
GET /admin/extension/vimServer/{id}/vmsList
Description:
Retreive a list of virtual machines that can be imported from a registered vCenter server.

This request supports optional query paramters:

  • page - The page to return. The first page is page 1.
  • pageSize - The number of virtual machines to list on a page. An integer value between 1 and 100. The default value is 1.
