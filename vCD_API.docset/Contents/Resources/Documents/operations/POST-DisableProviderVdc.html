POST /admin/extension/providervdc/{id}/action/disable

Operation:
POST /admin/extension/providervdc/{id}/action/disable
Description:
Disable a provider vDC.

You can disable a provider vDC to prevent the creation of organization vDCs that use its resources. When you disable a provider vDC, you also disable the organization vDCs that use its resources. Running vApps and powered on virtual machines continue to run in those vDCs, but users cannot deploy additional vApps in them.
