GET /admin/extension/object/{id}/selectorExtensions

Operation:
GET /admin/extension/object/{id}/selectorExtensions
Description:
Displays a list of all selector extensions of a registered object extension.
Since:
16.0
Input parameters
Consume media type(s):
None
Input type:
None
Output parameters
Representation of a collection of {@link SelectorExtensionType} associated with a. object extension

Produce media type(s):
application/vnd.vmware.admin.extensibility.selectors+xml
Output type:
SelectorExtensionsType
