GET /admin/extension/hostReferences/query

Operation:
GET /admin/extension/hostReferences/query
Description:
Get list of hosts by using REST API general QueryHAndler; This is read only list of host references and is not bound to specific states. If filter is provided it will be applied to the corresponding result set. Format determines the elements representation - referenceView or simpleView. Default format is referenceView.
