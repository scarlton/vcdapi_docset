GET /vAppTemplate/{id}/virtualHardwareSection

Operation:
GET /vAppTemplate/{id}/virtualHardwareSection
Description:
Retrieve the virtual hardware section of a VM.

This operation retrieves the entire VirtualHardwareSection of a VM. You can also retrieve many RASD item elements of a VirtualHardwareSection individually, or as groups of related items.

Since:
0.9
Input parameters
Consume media type(s):
None
Input type:
None
Output parameters
ovf:VirtualHardwareSection_Type

Produce media type(s):
application/vnd.vmware.vcloud.virtualHardwareSection+xml
Output type:
VirtualHardwareSection_Type
