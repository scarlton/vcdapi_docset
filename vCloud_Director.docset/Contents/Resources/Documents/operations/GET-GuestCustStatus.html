GET /vApp/{id}/guestcustomizationstatus

Operation:
GET /vApp/{id}/guestcustomizationstatus
Description:
Retrieve status of guest customization for that VM Following are the possible status 0 - PENDING_GC : Guest Customization is either not started or in progress. This is set whenever a VM need customization. 1 - PENDING_REBOOT : GC is successful, but reboot is pending. This is set when vCloud receives "guestinfo.gc.status" as successful 2 - PENDING_POST_GC : Reboot has happened, waiting for post gc script to complete. This is set when tools running status comes as "Running" based on conditions 1 & 2 3 - COMPLETE : GC is complete.
Since:
13.0
